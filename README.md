# Degen Token Implementation

An implementation of an assessment for metacrafters where a new token is to be launched in avalanche testnet 

## Description

A token that could be launched in remix.ethereum.org which can do the following things
- Minting: only the owner can mint tokens
- getBalance: Getting the balance of your own address
- transferTokens: transfering tokens from one address to another
- burnTokens: removing of tokens from your address
- redeemItem: redeems an item to the address based on the itemId passed
- checkItems: checks all the items available
- getUserItems: gets all of the items that the address currently has
- addItemToStore: adds a new item to the store. Only the owner is able to use this function.

## Getting Started

### Installing

* To use, upload the repository on [remix.ethereum.org](https://remix.ethereum.org/)
* Add your address to the *at address* and change the environment to *injected provider* and choose your corresponding wallet
* Deploy the contract
